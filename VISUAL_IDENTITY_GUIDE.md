# SkyUnit AI - Brand Identity Guide
# دليل الهوية البصرية لمنصة سكاي يونيت الذكاء الاصطناعي

## الألوان الأساسية | PRIMARY COLORS

### الأزرق | BLUE PALETTE
- **Dark Blue**: #0A3F5E (الأزرق الحلبي العميق)
- **Primary Blue**: #0B5BA8 (الأزرق الأساسي)
- **Medium Blue**: #1E7DB8 (الأزرق المتوسط)
- **Light Blue**: #2B9FD9 (الأزرق الفاتح)
- **Sky Blue**: #4DBFE8 (الأزرق السماوي)

### النصوص | TEXT COLORS  
- **White**: #FFFFFF (أبيض نقي)
- **Off-White**: #F8FAFB (رمادي فاتح)

## التدرجات | GRADIENTS

### التدرج الرئيسي
```
Background Gradient:
From: #0A3F5E (Dark Navy Bottom)
Via: #0B5BA8 (Primary Blue)
To: #2B9FD9 (Light Blue Top)
```

### التدرج الزاهي
```
Accent Gradient:
From: #1E7DB8
To: #4DBFE8
With Glow Effect
```

## المكونات | COMPONENTS

### بطاقات المنتجات | PRODUCT CARDS
- Background: Linear gradient #0B5BA8 → #1E7DB8
- Border Radius: 12px
- Box Shadow: 0 8px 32px rgba(43, 159, 217, 0.3)
- Text Color: #FFFFFF
- Accent Text: #4DBFE8

### الأزرار | BUTTONS
- Primary BG: #2B9FD9
- Primary Hover: #1E7DB8  
- Primary Text: #FFFFFF
- Border Radius: 6px
- Shadow: 0 4px 12px rgba(43, 159, 217, 0.4)

### رؤوس | HEADERS
- Font: Bold, RTL Optimized
- Color: #FFFFFF
- Size: 24px - 48px
- Letter Spacing: 0.5px

## المسافات | SPACING
- Base Unit: 8px
- Padding: 16px, 24px, 32px
- Gap: 16px - 24px
- Margin: 32px, 48px

## العمق | DEPTH
- Shadow Light: 0 2px 8px rgba(0, 0, 0, 0.1)
- Shadow Medium: 0 4px 16px rgba(0, 0, 0, 0.15)  
- Shadow Heavy: 0 8px 32px rgba(43, 159, 217, 0.3)
- Glow: 0 0 20px rgba(77, 191, 232, 0.4)

## المراقبات | ANIMATIONS
- Transition: 0.3s ease-out
- Hover Scale: 1.02
- Glow Pulse: 1.5s infinite

## تطبيقات الالوان | COLOR APPLICATIONS

### Hero Section
- BG Gradient: #0A3F5E → #0B5BA8 → #2B9FD9
- Title: #FFFFFF
- Subtitle: #4DBFE8

### Projects Grid
- Card BG: #0B5BA8 with border #2B9FD9

### Table
- Header BG: #0B5BA8

### Form Elements

## قواعد الاتساق | CONSISTENCY RULES

✅ DO's:
- Use only specified blue palette
- Apply gradients consistently
- Maintain RTL text alignment
- Keep glow effects subtle
- Preserve spacing ratios
- Test on all screen sizes

❌ DON'Ts:
- Mix with warm colors (red, orange)
- Use pure black text on dark backgrounds
- Override font weights
- Add heavy shadows
- Change border radius dramatically
- Ignore spacing guidelines
